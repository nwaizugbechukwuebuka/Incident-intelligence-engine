# System Overview

The system ingests unstructured incident data, processes it with LLMs, and automates response workflows via n8n. It is designed for extensibility, security, and real-world deployment.

## Key Features
- Modular Python backend (FastAPI)
- LLM integration (OpenAI, Azure, etc.)
- n8n workflow automation
- Structured logging and error handling
- CI/CD and containerization
